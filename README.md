# The Antecedent

> *Showing up with kindness, on purpose.*

A resource and encouragement app built for Behavior Technicians (BTs), Registered Behavior Technicians (RBTs), BCaBAs, and BCBAs in Applied Behavior Analysis.

Inspired by 1 John 3:17–18 — love in action, not just words.

---

## What's inside

- **Home** — Daily affirmations, mood check-in, kindness notes, and today's message
- **Learn** — Message library (20 notes) and interactive BACB-aligned study modules for BT, RBT, BCaBA, and BCBA credential paths
- **Support** — Guided conversation frameworks for difficult workplace situations and career guidance
- **Resources** — Curated links to BACB resources, essential books, YouTube channels, and professional development
- **Me** — Personal profile, points system, badges, and monthly progress summary

---

## Tech stack

- React 18
- Vite
- Google Fonts (Playfair Display, Lora, DM Sans)
- No external UI dependencies

## Running locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploying

This project is configured for one-click deployment on [Vercel](https://vercel.com).

Connect your GitHub repository to Vercel and it will detect the Vite/React setup automatically.
